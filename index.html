                • OZU Game Jam | Run From Pencil Case! | Unity (2D) / C# / Aseprite (PC)

              About the Game
              As part of a two-day game jam event with the theme 'Start Small', we created a 2D platformer game featuring an eraser attempting to escape from a pencil case. The player must navigate through obstacles by throwing the eraser's pens as spears to create platforms to jump on and climb over.
              The game presents a unique twist where letters must be uncovered by the eraser to solve a riddle at the game's end. These letters serve as checkpoints, allowing the player to respawn in a particular letter if the pencil case catches them.

              What I did in this group of four project?
            • • Developed all of the game mechanics and features using C#
            • • Calibrated most of pre-made PNG animations in Unity
            • • Adjusted various aspects of the game in Unity editor
            • • Contributed idea of gameplay and the concept
            • • Designed user interface
